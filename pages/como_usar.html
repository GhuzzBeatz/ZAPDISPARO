1. Conectar o WhatsApp
1
Acesse Conexão WhatsApp → clique em Conectar → aguarde o QR Code aparecer.
2
No celular: WhatsApp → 3 pontinhos → Aparelhos Conectados → Aponte para o QR Code.
✓
A sessão é salva — na próxima vez reconecta automático.
2. Importar Contatos via Excel ou CSV
1
Prepare uma planilha Excel ou CSV com as colunas: telefone (obrigatório), nome, empresa, produto, valor, cidade.
2
Em Novo Disparo clique em Importar Excel / CSV e selecione o arquivo. Os contatos são carregados automaticamente.
3
Telefones com menos de 8 dígitos são ignorados automaticamente.
3. Escrever a Mensagem com Variáveis
1
Use variáveis para personalizar cada mensagem: {nome}, {empresa}, {produto}, {valor}, {cidade}.
2
A prévia ao lado mostra como a mensagem ficará para o primeiro contato da lista.
✓
Salve como Modelo para reutilizar em futuros disparos.
4. Configurar o Delay e Disparar
1
Configure o delay entre mensagens. Recomendamos entre 8 e 15 segundos para evitar bloqueios.
2
Clique em 🚀 Iniciar Disparo. O log em tempo real mostra cada envio.
⏸
Use Pausar para interromper temporariamente e Parar para cancelar.
⚠️ Evite enviar mais de 200 mensagens por hora para não ser bloqueado pelo WhatsApp.
